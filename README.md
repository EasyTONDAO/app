# near-dao
DAOs on NEAR
VueJS application

## Requirements
MDB Vue Advanced UI KIT (https://mdbootstrap.com/docs/b5/vue/#!)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
for testnet
```
npm run build_mainnet
```
for mainnet

### Lints and fixes files
```
npm run lint
```
